# Scripts

Thư mục chứa các script tiện ích cho dự án.

## Các script có sẵn:

- `setup.sh` - Cài đặt và khởi tạo dự án
- `start.sh` - Khởi động các service
- `info.sh` - Hiển thị thông tin hệ thống
- `generate-config.sh` - Tạo file cấu hình
- `change-port.sh` - Thay đổi port

## Sử dụng:

```bash
# Chạy từ thư mục gốc
bash scripts/setup.sh
bash scripts/start.sh
```
